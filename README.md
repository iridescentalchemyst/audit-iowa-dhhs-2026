# Petition to Audit Iowa DHHS — 2026

**Petitioner:** Ashley Marie Meredith, MSN — Pro Se  
**Filed:** Iowa State Auditor's Office  
**Status:** In Progress — v2.0 Preliminary Draft  

---

## What This Is

A citizen-initiated audit petition submitted to the Iowa State Auditor under Iowa Code §§ 11.6 and 11.24, supported by a 23-year longitudinal evidentiary record documenting systemic failures in Iowa's child welfare system across four rounds of federal Child and Family Services Reviews (CFSRs).

This repository hosts the public transparency version of the petition packet, including all 20 petition sections, the evidentiary attachment index, open records request logs, and the authority outreach record.

---

## Document Architecture

The petition uses a **three-document architecture**:

| Document | Purpose |
|---|---|
| **Petition to Audit** (~150–175 pp.) | Primary filing — GAGAS-informed, 20 sections |
| **CFSR Companion Report** | Longitudinal analysis — Rounds 1–4 + Round 4 deep dive |
| **CCWIS Companion Report** | IT procurement, transparency, and data integrity |

## Evidentiary Attachments

- All four CFSR complete records (Rounds 1–4)

- Auditor of State special investigations

- Open records request communication logs (6 active ORRs)
